[Сравнение углов, выраженных в градусах и радианах. ](https://ru.wikipedia.org/wiki/Тау_(математика)) 
Диаграмма, показывающая соотношение между пи, тау и радианами с помощью окружности. Показывает преобразование градусов в радианы, а также знаки основных тригонометрических функций в каждом квадранте.

[Тау — это просто пи, умноженное на 2? Пи — это отношение длины окружности к диаметру. 
Тау же — это отношение длины окружности к радиусу; проще говоря, это удвоенное значение числа пи (τ = 2π).](https://www.google.com/search?q=tau+%3D+pi+*+2&sourceid=chrome&ie=UTF-8)
